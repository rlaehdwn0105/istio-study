# Week 1 - Istio 소개, 첫걸음

작성일: 2026-04-14

이 디렉터리는 Istio 스터디 1주차 학습 자료를 정리한 패키지다. 목표는 단순 링크 수집이 아니라, 나중에 그대로 git에 올려서 다시 학습할 수 있는 형태로 주차별 자료를 축적하는 것이다.

## 구성

- `notes/week1-study-guide.md`: 1주차 학습 가이드와 체크리스트
- `notes/week1-concepts.md`: 개념 정리 노트
- `notes/week1-source-comparison.md`: 블로그/공식 문서 비교 정리
- `sources/`: 원문 HTML 보관본
- `sources/INDEX.md`: 수집한 출처 목록과 상태
- `assets/manifests/images.md`: 이미지 자산 출처 목록
- `assets/raw/`: 실제로 내려받은 일부 공개 이미지
- `practice/`: 실습 YAML과 실행 가이드

## 이번 주 핵심 질문

- Istio를 왜 쓰는가
- Service Mesh는 무엇을 애플리케이션 바깥으로 분리하는가
- Envoy는 어디서 어떤 역할을 하는가
- Istiod는 어떤 설정을 내려주는가
- 첫 번째 애플리케이션을 메시에 붙이면 무엇이 달라지는가

## 소스 선택 원칙

- 공식 개념과 설치 절차는 `istio.io`를 기준으로 삼는다
- 한국어 설명과 도식화는 참여자 블로그를 참고한다
- 오래된 버전 실습은 흐름 이해용으로 보고, 실제 실습 파일은 최신 공식 릴리스를 우선한다

## 버전 주의

- 참여자 스터디 글 중 다수는 `Istio 1.17.8` 기준으로 작성되었다
- 현재 학습 자료는 `최신 공식 문서`와 함께 읽는 것을 전제로 한다
- 예제 YAML 필드와 동작은 버전에 따라 다를 수 있다
